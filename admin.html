How to update pricing

1. Edit any plan name, price, tagline, or features above.
2. Click Save & Deploy — this saves pricing.json to your repo.
3. The website automatically reads from pricing.json — no HTML changes needed.
4. Changes go live in ~30 seconds after save.

To add a new feature: Click "+ Add feature" under any plan.
To remove a feature: Click the red × next to it.
To reorder plans: Not supported yet — edit pricing.json directly for now.
